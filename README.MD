# Arid Next

## Project Description
Arid Next is a web application built with Next.js. This project leverages a variety of React components and libraries to create a dynamic and responsive user experience.

## Getting Started

### Prerequisites
- Node.js (>= 20.19.0)
- npm (>= 10)

Tip: this repo includes a `.nvmrc`, so you can run:

```sh
nvm install
nvm use
```


Optional (recommended for teams): Volta

If you use Volta (https://volta.sh), this repo pins Node/npm versions in `package.json`.
Just install Volta once and it will automatically use the right toolchain when you enter the project.

This repo also includes a `packageManager` pin (npm version) for tooling/CI.
